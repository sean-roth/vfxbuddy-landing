# VFX Buddy Landing Page

## Add your images

Upload these to the `images/` folder:
- `VB_Man_preview.png`
- `VB_Kitten_preview.png`
- `VB_Birds_preview.png`
- `VB_Dress_preview.png`

## Update links

- Replace `YOUR_CALENDLY_LINK` in index.html with your actual Calendly URL
- Update email if different from sean@vfxbuddy.ai

## Deploy to Vercel

1. Go to vercel.com
2. Sign in with GitHub
3. Import this repo (sean-roth/vfxbuddy-landing)
4. It auto-deploys
5. Add custom domain: vfxbuddy.ai

## Local preview

Just open index.html in a browser. No build step needed.
